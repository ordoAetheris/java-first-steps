# Задания: Файлы (I/O — Input/Output, ввод/вывод)

## Задание 1: Записать и прочитать ★☆☆
Файл `Task01.java`. Записать строку в `output.txt` (`Files.writeString`). Прочитать обратно (`Files.readString`). Вывести.
```bash
java Task01
# Записано в output.txt: 'Привет из Java!'
# Прочитано: 'Привет из Java!'
```

## Задание 2: Построчное чтение с номерами строк ★☆☆
Файл `Task02.java`. Имя файла из args. Читать через BufferedReader, выводить с номерами.
```bash
java Task02 poem.txt
# 1: Я помню чудное мгновенье
# 2: Передо мной явилась ты
```

## Задание 3: Копирование файла ★★☆
Файл `Task03.java`. `java Task03 source.txt copy.txt`. Побайтовое копирование через InputStream → OutputStream.

## Задание 4: Парсер логов — найти ошибки ★★☆
Файл `Task04.java`. Прочитать лог-файл. Строки содержащие "ERROR" → записать в `errors.txt`.

## Задание 5: CSV — чтение и обработка ★★☆
Файл `Task05.java`. Прочитать `data.csv` (имя,возраст,город). Вывести таблицей. Посчитать средний возраст.

## Задание 6: Properties (файл настроек) ★★☆
Файл `Task06.java`. Прочитать `config.properties` (db.host=localhost, db.port=5432). Вывести все настройки.

## Задание 7: try-with-resources — сравнение ★★★
Файл `Task07.java`. Два варианта одного кода: с `finally { reader.close(); }` и с `try (BufferedReader reader = ...)`. Второй — короче и безопаснее (не забудешь закрыть).

## Задание 8: Кофейня — сохранение и загрузка заказов ★★★
Файл `Task08.java`. Сохранить List<Order> в CSV-файл. Загрузить обратно. Данные переживают перезапуск программы.

## Задание 9: Входной и выходной файл из аргументов ★★☆
`java Task09 input.txt output.txt` — прочитать, обработать (удалить пустые строки), записать.

## Задание 10: Размер директории (папки) — рекурсия ★★★
Файл `Task10.java`. `java Task10 /path/to/dir`. Рекурсивно (заходя во вложенные папки) подсчитать суммарный размер всех файлов.
