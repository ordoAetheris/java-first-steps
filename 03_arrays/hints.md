# Подсказки: Массивы

## Задание 1: Сумма — for цикл, переменная-аккумулятор sum += arr[i]
## Задание 2: Min/Max — начать с arr[0], сравнивать каждый элемент
## Задание 3: Реверс — swap: `int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;` i от 0, j от length-1
## Задание 4: indexOf — цикл, if (arr[i] == target) return i; в конце return -1
## Задание 5: Подсчёт — два массива или вложенный цикл (пока нет Map)
## Задание 6: Bubble sort — два вложенных цикла, swap если arr[i] > arr[i+1]
## Задание 7: b = a — это ССЫЛКА. b[0] = 999 меняет и a[0]. Для копии: Arrays.copyOf(a, a.length)
## Задание 8: char[][] board = new char[3][3]. Проверка: 3 вложенных if (ряды, столбцы, диагонали)
## Задание 9: Integer.parseInt(args[i]) в цикле → int[], потом bubble sort
## Задание 10: Два массива одного размера. names[i] соответствует prices[i]
