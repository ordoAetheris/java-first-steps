# Задания: Массивы

## Задание 1: Сумма и среднее ★☆☆
Файл `Task01.java`. Дан массив `{10, 25, 33, 14, 7, 88, 42}`. Вычислить сумму и среднее.

## Задание 2: Минимум и максимум ★☆☆
Файл `Task02.java`. Найти минимальный и максимальный элемент в массиве. Без `Arrays.sort`.

## Задание 3: Реверс ★★☆
Файл `Task03.java`. Развернуть массив на месте (без второго массива). `{1,2,3,4,5}` → `{5,4,3,2,1}`.
Подсказка: swap первый↔последний, второй↔предпоследний...

## Задание 4: Поиск элемента ★☆☆
Файл `Task04.java`. Написать метод `int indexOf(int[] arr, int target)` — вернуть индекс элемента или -1 если нет.

## Задание 5: Подсчёт вхождений ★★☆
Файл `Task05.java`. Массив из аргументов командной строки (числа). Посчитать сколько раз встречается каждое число.
```bash
java Task05 1 3 2 1 3 1 4 2
# 1: 3 раза
# 2: 2 раза
# 3: 2 раза
# 4: 1 раз
```

## Задание 6: Сортировка пузырьком ★★☆
Файл `Task06.java`. Реализовать bubble sort вручную. Вывести массив до и после.
Не используйте `Arrays.sort` — смысл написать самому.

## Задание 7: Ссылочная ловушка ★★★
Файл `Task07.java`. Что выведет? Подумайте ПЕРЕД запуском.
```java
public class Task07 {
    public static void main(String[] args) {
        int[] a = {1, 2, 3};
        int[] b = a;          // копия или ссылка?
        b[0] = 999;
        System.out.println("a[0] = " + a[0]); // ???
    }
}
```

## Задание 8: Двумерный массив — крестики-нолики ★★★
Файл `Task08.java`. Поле 3×3 (`char[][] board`). Заполнить X и O. Вывести поле. Проверить есть ли победитель (три в ряд по горизонтали, вертикали или диагонали).

## Задание 9: Массив из аргументов + сортировка ★★☆
Файл `Task09.java`. Числа из args → массив int[] → сортировка → вывод.
```bash
java Task09 42 7 15 3 99 1
# До: [42, 7, 15, 3, 99, 1]
# После: [1, 3, 7, 15, 42, 99]
```

## Задание 10: Кофейня — меню ★★★
Файл `Task10.java`. Два параллельных массива: `String[] names`, `double[] prices`. Вывести меню, найти самый дорогой/дешёвый, посчитать средний чек.
