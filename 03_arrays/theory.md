# Массивы

## Что это

Массив — фиксированный набор элементов одного типа. Как ряд пронумерованных ячеек.

```java
int[] numbers = new int[5];        // 5 ячеек, заполнены нулями
numbers[0] = 10;                   // первый элемент (нумерация с 0!)
numbers[4] = 50;                   // последний (не 5!)

int[] scores = {90, 75, 88, 62, 95}; // сразу с значениями
System.out.println(scores.length);     // 5 (размер массива)
```

**Нумерация с нуля.** `scores[0]` = 90, `scores[4]` = 95. `scores[5]` → `ArrayIndexOutOfBoundsException (выход за границы массива)` (вылет).

## Обход

```java
// Классический for
for (int i = 0; i < scores.length; i++) {
    System.out.println("Элемент " + i + ": " + scores[i]);
}

// for-each (когда индекс не нужен)
for (int score : scores) {
    System.out.println(score);
}
```

## Важно

- Размер массива **фиксирован**. Создали 5 — будет 5. Нужно больше? Создавайте новый.
- `int[]` заполняется нулями. `String[]` — null'ами. `boolean[]` — false.
- Массив — это **ссылка**. `int[] b = a` — это НЕ копия, это вторая ссылка на тот же массив.
