# Переменные и типы данных

## Что это

Переменная — единица хранения информации. Коробка с наклейкой (именем), в которой лежит значение.

```java
int age = 28;          // целое число
double salary = 150000.50; // дробное число
String name = "Павел";     // текст
boolean employed = true;   // да/нет
```

## Зачем разные типы

Компьютер хранит данные в памяти. Память — это байты. Разные типы данных — разные размеры:

| Тип | Размер | Диапазон | Зачем |
|-----|--------|----------|-------|
| `byte` | 1 байт | -128 ... 127 | Маленькие числа, сырые данные |
| `short` | 2 байта | -32768 ... 32767 | Редко используется |
| `int` | 4 байта | -2.1 млрд ... 2.1 млрд | **Основной тип для целых** |
| `long` | 8 байт | ±9.2 квинтиллиона | ID в базе данных, timestamps |
| `float` | 4 байта | ~7 знаков точности | Редко (неточный) |
| `double` | 8 байт | ~15 знаков точности | **Основной тип для дробных** |
| `char` | 2 байта | Один символ | 'A', 'Б', '9' |
| `boolean` | 1 бит* | true / false | Да/нет, вкл/выкл |
| `String` | зависит | Текст любой длины | **Не примитив — объект** |

Зачем не хранить всё в `long`? Потому что 1 миллион переменных × 8 байт = 8 МБ. А если `byte` — то 1 МБ. В масштабе серверов, обрабатывающих миллионы запросов — это разница.

## Объявление и инициализация

```java
// Объявление: "создай коробку типа int с именем count"
int count;

// Инициализация: "положи в коробку значение 0"
count = 0;

// Одной строкой:
int count = 0;

// Несколько переменных одного типа:
int x = 1, y = 2, z = 3;

// Константа (нельзя менять после присвоения):
final double PI = 3.14159;
```

## Операции

```java
int a = 10, b = 3;

int sum = a + b;      // 13
int diff = a - b;     // 7
int product = a * b;  // 30
int quotient = a / b; // 3 (!) — целочисленное деление, дробная часть ОТБРОШЕНА
int remainder = a % b; // 1 — остаток от деления

double precise = (double) a / b; // 3.333... — приведение типа
```

**Важно:** `10 / 3 = 3`, не `3.333`. Целое делённое на целое = целое. Это ловит всех новичков.

## Преобразование типов

```java
// Автоматическое (безопасное: маленький → большой)
int x = 42;
long big = x;        // OK: int → long
double d = x;        // OK: int → double (42.0)

// Явное (опасное: большой → маленький, можно потерять данные)
long big = 3_000_000_000L;
int x = (int) big;   // ОБРЕЗАНИЕ! Результат: -1294967296 (переполнение!)
```

## Напоминалка: компиляция и запуск

```bash
# Написать код
nano Task01.java

# Скомпилировать
javac Task01.java

# Запустить
java Task01

# Запустить с аргументами
java Task01 arg1 arg2
```

Каждое задание — отдельный `.java` файл. Имя файла = имя класса.
