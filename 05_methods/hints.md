## 2: Високосный: (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)
## 3: average = (double)sum(arr) / arr.length — не забыть cast!

## 7: Примитивы (int) — по значению (копия). Объекты/массивы — по ссылке
## 8: Каждый метод делает ОДНУ вещь. main() только оркестрирует
## 9: Random.nextInt(max - min + 1) + min — число в диапазоне
