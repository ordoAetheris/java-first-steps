# Задания: Строки (String — текстовые данные)

## Задание 1: Реверс (переворот) строки ★☆☆
Файл `Task01.java`. Принимает строку из args (аргументов). Выводит задом наперёд. БЕЗ `StringBuilder.reverse()` — через цикл и `charAt()`.
```bash
java Task01 hello    # olleh
java Task01 Java     # avaJ
```

## Задание 2: Подсчёт слов и символов ★☆☆
Файл `Task02.java`. Текст из аргументов (слова через пробел). Вывести: количество слов, количество символов (без пробелов), самое длинное слово.
```bash
java Task02 "Java is a programming language"
# Слов: 5, Символов: 26, Самое длинное: programming (11)
```

## Задание 3: Цензура ★★☆
Файл `Task03.java`. Массив "плохих" слов. Заменить каждое на `***` в тексте (без учёта регистра — case-insensitive).
```bash
java Task03 "This is a bad and ugly example"
# This is a *** and *** example
```
Подсказка: `text.toLowerCase().replace(badWord, "***")` — но теряет регистр оригинала. Как сохранить?

## Задание 4: CamelCase ↔ snake_case конвертер ★★☆
Файл `Task04.java`. Два режима: `camel` и `snake`.
```bash
java Task04 camel my_variable_name     # myVariableName
java Task04 snake myVariableName       # my_variable_name
```
Подсказка для camelToSnake: если символ заглавный → вставить `_` перед ним + привести к строчному.

## Задание 5: Парсер (разбор) CSV ★★☆
Файл `Task05.java`. Строка CSV (значения через запятую): `"Павел,28,Новосибирск,Java"`. Разбить и вывести таблицей:
```
Поле 1: Павел
Поле 2: 28
Поле 3: Новосибирск
```

## Задание 6: Анаграммы (слова из одних букв) ★★☆
Файл `Task06.java`. Два слова из args. Проверить — анаграммы ли (одинаковые буквы в разном порядке).
```bash
java Task06 listen silent    # Да, анаграммы!
java Task06 hello world      # Нет, не анаграммы
```
Подсказка: отсортировать буквы обоих слов, сравнить.

## Задание 7: Генератор паролей ★★☆
Файл `Task07.java`. Аргумент: длина пароля. Генерирует случайный пароль из букв (маленьких + больших), цифр, спецсимволов.
```bash
java Task07 12    # aK3$mN7!pQ2x
java Task07 8     # Bx9#kL2p
```

## Задание 8: StringBuilder vs += — замер скорости ★★★
Файл `Task08.java`. Конкатенация (склеивание) 100 000 строк двумя способами. Замерить время.
```bash
java Task08
# String +=:     3200 мс (миллисекунд)
# StringBuilder:    5 мс
# Разница: 640x
```
**Подковырка:** `+=` создаёт НОВУЮ строку на каждой итерации → O(n²) по памяти. StringBuilder — один буфер → O(n).

## Задание 9: Шаблонизатор (template engine) ★★★
Файл `Task09.java`. Шаблон (template): `"Привет, {name}! Тебе {age} лет, ты из {city}."`. Подстановка значений.
```bash
java Task09
# Привет, Павел! Тебе 28 лет, ты из Новосибирска.
```
Подсказка: `template.replace("{name}", "Павел")` для каждого ключа.
