# Задания: Коллекции (Collections — умные контейнеры для данных)

## Задание 1: Список покупок ★☆☆
Файл `Task01.java`. `ArrayList<String>`: добавить 5 продуктов, удалить один, проверить наличие, вывести все.
```bash
java Task01
# [молоко, хлеб, сыр, масло, яблоки]
# Удаляем 'масло': [молоко, хлеб, сыр, яблоки]
# Содержит 'хлеб'? true
```

## Задание 2: Уникальные слова в тексте ★☆☆
Файл `Task02.java`. Текст из args → `HashSet<String>` (набор без дубликатов) → вывести количество уникальных.
```bash
java Task02 "the cat sat on the mat the cat"
# Всего слов: 8, Уникальных: 5 (the, cat, sat, on, mat)
```

## Задание 3: Частота слов ★★☆
Файл `Task03.java`. Текст → `HashMap<String, Integer>` (словарь: слово → количество). Какое самое частое?
```bash
java Task03 "the cat sat on the mat the cat"
# the: 3, cat: 2, sat: 1, on: 1, mat: 1
# Самое частое: 'the' (3 раза)
```

## Задание 4: Телефонная книга ★★☆
Файл `Task04.java`. `Map<String, List<String>>` — имя → список номеров (у человека может быть несколько). Добавление, поиск, удаление номера.

## Задание 5: Очередь (Queue) в банке ★★☆
Файл `Task05.java`. `Queue<String>` (LinkedList как очередь). Клиенты приходят (offer — добавить в конец), обслуживаются (poll — взять первого), заглянуть кто следующий (peek).

## Задание 6: Сортировка объектов ★★★
Файл `Task06.java`. `List<Student>`. Сортировка по имени: `Comparator.comparing(Student::getName)`. По оценке: `Comparator.comparing(Student::getGrade).reversed()` (от лучшей к худшей).

## Задание 7: Удаление дубликатов с сохранением порядка ★★☆
Файл `Task07.java`. List с дубликатами → `LinkedHashSet` (набор, сохраняющий порядок вставки) → обратно в List. Порядок первого появления сохраняется.

## Задание 8: Группировка заказов ★★★
Файл `Task08.java`. Список заказов → `Map<String, List<Order>>` по категории. Ручной цикл (ещё не знаем Stream API).

## Задание 9: Ловушка — удаление при итерации (обходе) ★★★
Файл `Task09.java`. Удалить элемент из List внутри for-each цикла → `ConcurrentModificationException`! Показать ошибку, потом — правильный способ через `Iterator.remove()`.

## Задание 10: Кофейня — статистика по дням ★★★
Файл `Task10.java`. `Map<String, List<Order>>`: дата (строка) → список заказов. Вывести: сумма за каждый день, средний чек, самый прибыльный день.
